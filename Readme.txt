

== 1. Texlive !
To use it, you need 

apt-get install --no-install-recommends texlive
apt-get install texlive-font-utils 


== 2. Powerful editor such as nedit and graphic creator such as xfig !
apt-get install make 
apt-get install nedit xfig 


== 3. How to use

make 
make view



Voila, Done, Enjoy !
